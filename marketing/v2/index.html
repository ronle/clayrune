Most AI tooling assumes you're doing one heads-down session. Clayrune assumes you're running 5–20 streams in parallel — research, drafts, refactors, scrapes — and that the primary problem is keeping track of what each one is doing, where each one got stuck, and which ones need you next.

The three things no one else does.

01 / Multi-project dashboard

Many agents, one operator console.

Open 5–20 project modals at once, drag them around your screen, watch them all stream. Per-project locks, per-project memory, per-project session history. Status pills surface what needs you first — asking, blocked, stuck — before everything else. No direct competitor at this positioning.

02 / Persistence

Sessions that actually survive.

Crash the app, reboot the laptop, lose the tab. Clayrune auto-revives sessions from its own log and from Claude's own JSONL transcripts on disk. 24-hour stale-session window. "Continue" button on every past run. The conversation comes back where you left it.

03 / Plan-approval gate

The tool that asks before it acts.

When an agent emits a plan, Clayrune collapses it into Approve / Collapse buttons. Nothing dangerous runs without your click. The opposite of autonomous-agent design — counter-positioning vs. Devin and the "let it loose, hope for the best" school.

▾ Plan: refactor scanner-live
Move shared SQL into util/queries.py
Update both services to import from there
Add type hints to query signatures
Run black + ruff; verify green
Commit with descriptive message

And also what makes it ship-able day-to-day.

04 / Mobile + remote access

Restart the server from your phone.

Cloudflare tunnel + named devices put your dashboard at clayrune.io from any phone or laptop. Watch agents stream remotely, send follow-ups, approve plans. Including a remote server-restart button after deploying a fix — so you don't need to walk back to the desktop just to bounce the process.

05 / Two-tier memory

Memory that curates itself.

Per-project CLAUDE.md + MEMORY.md with auto-condense. When memory gets stale, a housekeeping agent compacts it in the background — no manual pruning. Cross-project shared rules + per-project overrides. The agent always knows what was decided yesterday, last week, and three projects ago.

CLAUDE.md
Use psycopg2 raw SQL · we don't trust ORMs here
Tests under tests/integration/, not unit/

2026-04-22: GCP Cloud SQL = fr3-pg (shared with FL3 — careful)
MEMORY_ARCHIVE.md · auto-condensed weekly
06 / First-class backlog

Tasks linked to the agents that ran them.

Each backlog item tracks priority, status, source — and links back to the agent session that worked it. Cross-project view lets you triage everything in one list. GitHub Issues sync. The progress bar on each project tile is real, not decorative — it's actual backlog completion.

Where Clayrune fits.

Honest comparison. Clayrune wins on operator-grade orchestration; loses on single-task depth.

vs. Wins on Loses on
Claude CLI Multi-project, persistence, mobile, plan approval, backlog Freshness, portability, official support
Cursor / Windsurf Project-agnostic, multi-project, scheduler, hivemind Editor-depth, inline diff, autocomplete
Devin Local infra, plan-approval gate, multi-project oversight Sandboxed compute, built-in browser
Claude / ChatGPT Desktop Multi-project, persistent sessions, scheduler, backlog Polish, native OS integrations
Aider / Cline / Continue.dev Multi-project orchestration, mobile remote, scheduler Smaller ecosystem, no git-commit native flow
For
  • People running 5–20 long-lived AI work-streams on their own machine
  • Operators who want a dashboard, not another single-pane editor
  • Anyone who's lost a session to a crash and wants that not to happen
  • Users who want plan-gate safety before agents run shell commands
  • Developers, PMs, ops people, traders, researchers — non-coding work too
Not for
  • Cursor users who want autocomplete + inline diff in a coding session
  • People wanting an autonomous agent to "just figure it out"
  • Users who don't want anything running on their own machine
  • Single-task workflows that finish in under a few minutes

Install in one command.

Clones the repo, sets up a Python venv, installs Claude CLI if missing, creates a desktop shortcut, opens the dashboard. Tested on macOS 14+, Ubuntu 22.04+, Windows 11.

macOS / Linux Windows (PowerShell)
$ curl -sSL https://clayrune.io/install.sh | sh
