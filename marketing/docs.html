Install

Mission Control runs locally on macOS, Windows, and Linux. Head to the Download page to grab the right build.

# macOS (Homebrew)
brew install ronle/tap/mission-control

# Linux / manual
curl -sSL https://get.missioncontrol.sh | sh
Note: You'll need an API key from your AI provider (Anthropic, OpenAI, or a local model). Mission Control will prompt for this on first run.

Your first project

On first launch, Mission Control sets up a sample Birthday party planner project and walks you through a four-step tour. Poke around without worry — the sample is safe to dismiss any time.

  1. Pick an assistant from the starter pack, or start blank.
  2. Give your project a name and a one-line goal.
  3. Hit Start. The assistant begins on its own.
  4. When it needs you, its card turns amber.

Settings & theme

Open ⚙ Settings from the sidebar to change:

  • Theme — Soft dark or Warm
  • Accent color — Sunset, Rose, Lilac, Lagoon, Ink
  • Density — Cozy or Compact
  • Voice — Casual or Professional

Understanding statuses

Every card wears one of five statuses. That's it.

  • Working on it — the assistant is actively making progress
  • Needs you — it has a question or decision it can't make alone
  • Stuck — something broke; it needs help to recover
  • All done — finished, ready for your review
  • Resting — nothing on its plate right now

Answering questions

When a card turns amber, click the primary button. You'll see the question and a short set of choices. Pick one, add context if you want, and send. The assistant picks back up immediately.

Custom assistants

An assistant is a single YAML file. Drop it in ~/.mission-control/assistants/ and it appears in the starter pack.

name: Book Club Organizer
emoji: 📚
prompt: |
  You help me run a monthly book club.
  Pick books, schedule meetings, summarize discussions.
tools:
  - calendar
  - email

Advanced mode

Developers: toggle Advanced in Settings to unlock the original power-user dashboard — raw streams, plan approvals, token counters, schedules, memory files, and GitHub issues sync.
