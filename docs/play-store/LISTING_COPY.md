## Full description (4000 chars max)

```
Clayrune is the official Android client for your self-hosted Clayrune
Mission Control dashboard — a multi-project management surface for
Claude Code agents.

This app is a thin, authenticated WebView to YOUR server. There is no
Clayrune cloud, no shared backend, no account on our side. Every
request goes through the secure Cloudflare Tunnel you provisioned to
the Mission Control instance you run on your own machine.

ONE-TAP SETUP

• Open Settings → Pair Mobile App on your Clayrune dashboard
• Scan the QR code with this app
• You're in — no Cloudflare account, no service-token paste, no
  hand-typed credentials

WHAT YOU CAN DO

• Watch your Claude Code agents work in real time across all your
  projects
• Send follow-up prompts from your phone
• Approve plans, answer agent questions, pause runs
• Get push notifications when an agent needs your attention or
  completes a long task
• Voice-to-text input for prompts (optional, on-device)

PRIVACY

• Zero data collection. No analytics, no ads, no tracking.
• Your credentials are stored in the Android Keystore via
  EncryptedSharedPreferences — never transmitted anywhere except as
  request headers to your tunnel URL.
• The QR scanner runs in Google Play Services and never sends image
  data anywhere; the app itself doesn't request camera permission.
• Full privacy policy: https://clayrune.io/privacy

REQUIREMENTS

• A running Clayrune Mission Control instance on a desktop or server
  (download free at https://clayrune.io)
• Path B "Remote Access" enrolled (the dashboard walks you through
  this — one click)

OPEN SOURCE

The Mission Control backend is open source:
https://github.com/ronle/mission-control
```
